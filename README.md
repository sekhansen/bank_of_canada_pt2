# Bank of Canada — Machine Learning for Economics

Materials from a short course on machine learning methods for economics taught at the Bank of Canada by [Stephen Hansen](https://github.com/sekhansen).

## Part 1: Applied LLMs for Forecasting and Policy Research

Lectures and code on using large language models for text-based economic analysis.

- Lectures: [part1/lectures/](./part1/lectures/) — `lecture1.pdf`, `lecture2.pdf`, `lecture3.pdf`
- Code: [part1/code/](./part1/code/)
  - `01_llm_api.ipynb` — interacting with LLM APIs
  - `02_encoder.ipynb` — encoder models and embeddings
  - `03_classification.ipynb` / `04_classification.ipynb` — classification with LLM features

## Part 2: Neural Networks for Time Series and DSGE Estimation

Lectures and code on neural network methods for macroeconomic forecasting and structural estimation.

- Lectures: [part2/lectures/](./part2/lectures/) — `lecture1.pdf`, `lecture2.pdf`
- Code: [part2/code/](./part2/code/)
  - `1_jax_neural_networks.ipynb` — neural network basics in JAX
  - `2_neoclassical_growth_nn.ipynb` — solving the neoclassical growth model with neural nets
  - `nowcast_canada.ipynb` / `nowcast_sentiment.ipynb` — nowcasting applications
